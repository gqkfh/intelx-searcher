# IntelX searcher with ApiKey

> **With this Python tool, you can search in the IntelX database for results. The tool includes the ApiKey**

- **`Windows` Installation**

```text
- Download the .zip
- Extract it
```

- **`Linux` Installation**

```shell
git clone https://github.com/gqkfh/intelx-searcher
```

> **Execution**

```shell
python3 searcher.py
```
